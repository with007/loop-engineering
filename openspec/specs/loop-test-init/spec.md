# Loop Test Init

初始化/完善项目测试文档的 Claude Code skill，将通用模板产物定制为项目专属文档。

## Purpose

`/loop-test-init` skill 在项目中运行，通过深入理解项目结构，将 `loop setup` 生成的通用模板文档（VERIFY.md 和 TEST.md）定制为包含实际命令、参数和流程的项目专属文档。填补通用模板和项目实际情况之间的 gap。

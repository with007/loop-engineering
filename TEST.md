# 手动测试指南 — loop-engineering

> 项目类型: Python Server

## 启动项目

```bash
python -m <模块名>
```


## 测试清单


### 1. 测试检查

**步骤**:

1. 运行 `python -m pytest`

2. 确认所有测试通过


**预期**: 全部测试通过，无失败


### 2. 服务启动冒烟测试

**步骤**:

1. 运行 serve 命令启动服务

2. 在浏览器打开对应 URL

3. 确认页面正常加载


**预期**: 服务正常启动，页面可访问



## 测试完成后

- 停止服务
- 检查终端无异常日志
- 确认无文件残留或脏状态
